# Arbeitsplatz auswählen

## User Story

Als INNOQ-Mitarbeiter möchte ich einen verfügbaren Arbeitsplatz auswählen und bestätigen, damit ich den für mich passenden Platz reservieren kann.

## Beschreibung

INNOQ-Mitarbeiter besuchen das Büro nur etwa einmal pro Woche und haben keinen festen Arbeitsplatz. Damit sie planungssicher ins Büro kommen können, müssen sie vorab sehen, welche Arbeitsplätze an ihrem Standort verfügbar sind, und einen davon gezielt auswählen. Die Auswahl bildet die Grundlage für die anschließende verbindliche Buchung.

## Akzeptanzkriterien

- [ ] Es werden nur Arbeitsplätze angezeigt, die im gewählten Zeitraum verfügbar sind
- [ ] Ein bereits gebuchter Arbeitsplatz kann nicht ausgewählt werden
- [ ] Die getroffene Auswahl wird vor der Bestätigung deutlich hervorgehoben
- [ ] Die Auswahl kann vor dem Absenden wieder geändert werden
