# QA-3 · Erste Buchung ohne Schulung

**Qualitätsmerkmal:** Benutzbarkeit / Erlernbarkeit
**Priorität:** 3 von 5

**Szenario (Prosa):**
> Bei der Erstnutzung ohne vorherige Schulung möchte ein neuer INNOQ-Mitarbeiter
> einen Konferenzraum für den nächsten Tag buchen. Er findet den Buchungs-Kernpfad in
> der Calvin-Web-App selbsterklärend und **schließt die Buchung in ≤ 5 Minuten ab;
> 90 % der neuen Mitarbeiter schaffen dies ohne fremde Hilfe.**

| Baustein | Ausprägung |
|----------|------------|
| **Environment** | Erstnutzung ohne Schulung, neuer Mitarbeiter am eigenen Gerät |
| **Source** | Neuer INNOQ-Mitarbeiter ohne Vorkenntnisse |
| **Event** | Möchte erstmals einen Konferenzraum für den Folgetag buchen |
| **Artifact** | Calvin Web-App (SPA), Buchungs-Kernpfad (Standort → Raum → Zeit → Bestätigung) |
| **Response** | Mitarbeiter findet selbsterklärend den Weg und schließt die Buchung erfolgreich ab |
| **Measure** | Abschluss in ≤ 5 Minuten; 90 % ohne fremde Hilfe |

**Motivation:** Eine niedrige Einstiegshürde fördert die angestrebte hohe
Mitarbeiterakzeptanz und entspricht dem Vision-Prinzip „Einfachheit“.
